# CRM Velocy

Monorepo inicial do CRM Velocy, estruturado para suportar o MVP definido na documentacao do projeto.

## Estrutura

- `apps/frontend`: aplicacao web operacional e administrativa
- `apps/backend`: API principal, integracoes, atendimento e auditoria
- `packages/shared`: tipos e contratos compartilhados
- `docs`: documentacao de produto, arquitetura e QA

## Objetivo do MVP

- WhatsApp receptivo oficial
- triagem inicial por IA via n8n
- handoff para humano com historico completo
- inbox operacional
- dashboard operacional inicial
